🎆서울세계불꽃축제

1. 이용요금

  • 무료

2. 주변 볼거리

  • 이촌한강공원
  • 이촌로 72길 62
  • 노들섬
  • 용산구 이촌동 302-146
  • 망원한강공원
  • 마포구 망원제1동 마포나루길 467

3. 축제 안내

  • 일자 : 2025.10.05
  • 주소 : 서울특별시 영등포구 여의동로 330 (여의도동)
  • 문의 : 02-519-9778
  • 교통 : 일대가 매우 혼잡하니 대중교통 이용바랍니다.

4. 소개

＊ 한강공원을 화려한 불꽃으로 수놓는 세계불꽃축제는 매년 10월경 여의도 한강공원(63스퀘어 앞)에서 펼쳐집니다.
    매년 세계적인 수준의 불꽃 전문 기업들이 초청되어 여의도의 밤 하늘을 무대로 환상적인 불꽃 연출을 선보이며,
    주간에도 다채로운 부대행사가 진행됩니다.
    (주)한화가 자랑하는 '멀티미디어 불꽃쇼'는 불꽃과 음악, 레이저 연출이 결합된 아시아 최고 수준의 불꽃쇼입니다.
